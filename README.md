# STM32F4-Servo-Motor-Controller
 A minimalistic project for controlling servo motors using STM32F401 microcontroller.

# Requirements
 - Servo Motor MG995
 - STM32  Microcontroller Board.
 - STM32CubeMX Tool. 
 - STM32CubeIDE Development Environment.
 - USB-Mini USB adapter.
